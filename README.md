## Authentication Service - [Open API](https://spring-boot.davidcamelo.com/swagger-ui/index.html?urls.primaryName=auth-service)
[![Infra Deployment](https://github.com/DavidCamelo/auth-service/actions/workflows/deploy.yml/badge.svg)](https://github.com/DavidCamelo/auth-service/actions/workflows/deploy.yml)

This project is based on Spring Boot and provides authentication and authorization services for the application ecosystem. It utilizes Spring Security and JWT (JSON Web Tokens) for secure user authentication and access control.
